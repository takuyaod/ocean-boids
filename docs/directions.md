# プロジェクト方向性メモ

## スタイル

インベーダーゲーム風のデザインを採用する：

- 黒背景 + ネオン/蛍光色（緑・シアン・マゼンタ）
- ドット絵（ピクセルアート）のキャラクター
- CRTフィルター（スキャンライン・グロウ効果）
- アーケードフォント（Press Start 2P など）

## 方向性候補

| 方向性 | 内容 |
|--------|------|
| **純粋なBoidsビジュアル** | エイリアン型のドット絵が画面を群れで飛び回るだけ（インタラクションなし） |
| **インタラクティブ** | マウスカーソルを捕食者として、エイリアンが逃げ回る |
| **進化シミュレーション** | 個体が世代を経て群れの行動が変化していく |
| **ゲーム要素あり** | プレイヤーが群れの中心を操作・群れVS群れ |
